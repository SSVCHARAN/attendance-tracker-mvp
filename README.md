# Attendance Tracker MVP

This is a full-stack web application built to manage student attendance and teacher coordination in a simple and efficient way.

## 🚀 Features

- 🔐 Authentication (Teacher / Student login)
- 📊 Attendance Management (period-wise)
- 📢 Teacher Absence Alerts
- 💬 Messaging System (public + private)
- 📄 PDF Sharing between teachers and students
- 📈 Student Dashboard to view attendance

## 🛠 Tech Stack

Frontend:
- React (Vite)
- Tailwind CSS

Backend:
- Node.js (Express)
- SQLite
- JWT Authentication
- Socket.io (real-time updates)

## 📂 Project Structure


client/ → Frontend (React)
server/ → Backend (Express + DB)


## ▶️ Run Locally

### Backend

cd server
npm install
npm run dev


### Frontend

cd client
npm install
npm run dev


Open:

http://localhost:5173


## 🎯 MVP Scope

This project focuses on building a working minimum viable product with core functionality before adding advanced features.
